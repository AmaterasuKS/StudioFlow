# Data Disclaimer

All data included in this repository is fictional and provided strictly for development, testing, and demonstration.

## Includes
- Seeded user identities, names, emails, and password hashes
- Seeded studios, descriptions, rates, capacities, and locations
- Configuration placeholders and token examples

## Excludes
- Real customer data
- Real employee/admin identities
- Production credentials, API tokens, or private secrets

Do not use seeded identities or credentials in production environments.
